1. Acesse o site
Abra o navegador e vá até o site: Pokémon Zone
2. Encontre as imagens das cartas
No site, busque as imagens das cartas que você possui e das que você precisa para sua coleção.
Copie o link direto da imagem da carta.
3. Preencha os campos
No site de geração de imagens, cole o link das imagens nos respectivos campos:
Campo das cartas que você possui: Cole o link da carta que já tem.
Campo das cartas que faltam: Cole o link da carta que ainda precisa.
4. Gere a imagem
Após preencher ambos os campos, clique no botão "Gerar Imagem".
5. Baixe a imagem gerada
Quando a imagem for gerada, você verá uma prévia.
Clique no botão "Download" para baixar a imagem para o seu computador ou dispositivo.
